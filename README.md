````markdown
# 💻 E-Shop Client (Frontend)

This is the user interface for the E-Shop platform, built using React, Vite, and Tailwind CSS.

> 💡 **Note:** To launch the full application (Frontend + API), please follow the ordering instructions in the main [Root Launch Guide](../README.md).

## 🛠️ Standalone Development Setup

If you are working exclusively on the UI, make sure your backend is running independently, then execute:

```bash
# 1. Install dependencies
npm install

# 2. Run local server
npm run dev
```
````
